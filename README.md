## 실제 복권이 아닙니다.
* 꽝만 나오는 즉석복권이 짜증나서 만든 어플
* 복권, 소액으로 건전하게 즐길때 희망과 행복이 됩니다. 

### 복권, 과몰입 시 중독 될 수 있습니다.
* 대다수의 사람들에게 복권구매란 한 주간의 즐거운 희망을 심어주는 가벼운 오락 활동의 하나입니다.
